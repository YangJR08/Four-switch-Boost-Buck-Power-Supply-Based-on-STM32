# Non-Linear Cascaded Closed-Loop Controller for Buck-Boost DC-DC Converter

这是一个基于 **MATLAB/Simulink** 开发的高性能双向/四开关 **Buck-Boost DC-DC 变换器** 的数字闭环控制器。该控制器采用多速率级联（电压外环 + 电流内环）架构，并融合了电压增益前馈与多模式自动迟滞切换技术，全面支持全自动代码生成（兼容 `%#codegen`）。

------

## 核心控制原理

控制器基于四开关 Buck-Boost 变换器的理想电压变比传输模型进行设计：

$$\frac{V_{out}}{V_{in}} \approx \frac{D_{buck}}{D_{boost}}$$

控制架构集成了**直接电压变比前馈**与**级联双闭环微调**：

1. **主通道（前馈+变比PI）**：通过目标电压变比 $R_{ff} = \frac{V_{ref}}{V_{in}}$ 提供瞬态响应极快的控制主基准。
2. **串级通道（电压环-电流环）**：电压外环根据 $V_{out}$ 误差计算出最优的电感电流基准 $I_{L\_ref}$；电流内环对其进行快速跟踪，产生的修正量作为微调项叠加至主变比通道，确保稳态无静差并具备极强的抗负载扰动能力。

------

## 关键特性

- **真正的软启动（防止启动暴冲）**：启动阶段 $V_{ref}$ 从安全下限（`Vref_min`）通过设定的斜坡变化率（默认 3000 V/s）线性爬升至目标值，彻底消除了上电瞬间由于误差饱和导致的过冲（成功解决早期版本暴冲至 37V 的问题）。
-  **独立双积分器架构（零稳态纹波设计）**：彻底隔离了级联环路（$I_L$ 基准计算）与主传输比通道的积分状态变量（`vInt` 与 `ratioInt` 相互独立），有效避免了多环路拉扯引起的系统低频谐振与稳态抖动。
- **多模式平滑迟滞切换**：根据实时变比自动在 **Buck 降压（Mode 1）**、**直通/混合（Mode 2）**、**Boost 升压（Mode 3）** 之间切换，带有双向迟滞区间，防止在临界点发生模式震荡（Mode Hopping）。
-  **双层安全保护机制**：
  - **动态功率/电流限制**：基于 $I_{out\_limit} = \min(I_{user\_max}, \frac{P_{max}}{V_{out}})$ 实时限幅。
  - **严重过流故障锁存（Mode 4）**：当输出电流滤波值超过限流值的 1.8 倍时，立即触发保护锁存，强行切断 Buck 占空比并清除电流基准，待电流降至安全释放阈值后方可复位。
- **平滑滤波与变化率限制（Slew Rate Limit）**：对反馈信号进行一阶低通滤波降噪，并限制单步占空比最大突变量（`slew = 0.01`），显著降低开关纹波和数字抖动。

## 调参及高级优化指南

如果针对特定的硬件拓扑（L、C参数变化）或工况需要进一步优化系统表现，请参考以下调整方向：

### 1. 调整启动速度

在代码中，软启动电压爬升速率由以下语句控制：

Matlab

```
dv = 3000.0 * tsi; % 默认 3000 V/s
```

> - **如果希望启动更快**：可将 `3000.0` 调大（例如 `5000.0`），但需注意观察接近目标电压时是否有轻微超调。
> - **如果硬件 LC 较大导致依然有微幅过冲**：可将其调小（例如 `1500.0`），让电压上升更平缓。

### 2. 消除模式切换边界的轻微纹波

在特定工况下（例如 $V_{in} \approx V_{out}$ ），若在切换模式时产生轻微纹波，可在代码中定位到以下板块：

Matlab

```
if Mode == 1.0
    dsRef = min(max(dsRef + 0.01, duty_min), duty_max);
elseif Mode == 3.0
    dbRef = min(max(dbRef + 0.01, duty_min), duty_max);
```

> **优化操作**：这里的 `0.01` 是为了模式平滑过渡引入的轻微重叠偏置。如果发现稳态时出现等幅微小抖动，可将该值降低（如 `0.002`）或者直接修改为 `0.0`。

### 3. 抑制稳态低频抖动

如果系统在稳态时出现类似正弦波的低频控制抖动，通常是主通道变比积分过快导致的：

Matlab

```
KvI = max(0.10 * kiv, 1.0);
```

> **优化操作**：可以将比例系数 `0.10` 降低（例如降低至 `0.02`），让主变比通道的积分动作更加钝化，从而高度依赖级联环路和前馈，以此获得极其纯净的稳态波形。
